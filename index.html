Rhythm Pulse

An Interactive Audiovisual Experience

Project Outline

Rhythm Pulse is an interactive audiovisual installation that creates a dynamic relationship between sound, light, and motion. The project combines Arduino-based physical inputs, real-time audio synthesis, and visual feedback to create an immersive experience where users can manipulate sound and visuals simultaneously.

The core concept explores how different sensory inputs can control and influence various aspects of a generative audiovisual composition. By turning knobs, pressing buttons, and altering light levels, users can discover how their physical interactions transform into digital experiences.

Project Description

Rhythm Pulse creates an abstract audiovisual environment where geometric shapes respond to music generated in real-time. At its heart, the project explores the relationship between physical interaction and digital response through three main components:

Physical Interface

The physical interface consists of three input methods:

  • A potentiometer that controls the size of geometric shapes
  • A button that toggles between different audio modes
  • A photoresistor that responds to ambient light and controls the color palette

These inputs allow for intuitive interaction with the digital elements. As users turn the potentiometer, shapes grow and shrink; when they press the button, the audio changes character; when they cover the light sensor, colors shift across the spectrum.

Visual Environment

The visual environment consists of geometric shapes (circles, squares, and triangles) that respond to audio frequencies:

  • Shapes change size based on potentiometer input and bass frequencies
  • Shapes rotate based on mid-range frequencies
  • Shapes move based on treble frequencies
  • Colors change based on light sensor readings
  • Particles appear with each beat, creating a dynamic visual rhythm

Sound Generation

The audio component uses Tone.js to generate synthesized sounds:

  • A drone synthesizer provides a continuous tonal foundation
  • A melodic synthesizer plays patterns of notes that change with button presses
  • A percussion synthesizer creates rhythmic beats
  • Audio parameters are modulated in real-time based on user input
  • Frequency analysis drives both visual reactions and LED outputs

Technical Implementation

The project integrates three key technologies:

  1. Arduino: Handles physical inputs and outputs through analog and digital pins
  2. p5.js: Creates the visual environment and manages the overall application
  3. Tone.js: Generates and manipulates audio in real-time

Data flows bidirectionally between the systems:

  • Arduino sends sensor data to the p5.js application via serial communication
  • p5.js processes this data to control both visuals and audio parameters
  • Tone.js generates audio and provides frequency analysis
  • p5.js sends frequency data back to Arduino to control LEDs

Hardware Components

  • Arduino UNO
  • Potentiometer (250kΩ) connected to A0
  • Push button connected to D3
  • Photoresistor with voltage divider connected to A1
  • Three LEDs with 1kΩ resistors connected to pins D9, D10, and D11

Reflections

Creating Rhythm Pulse taught me valuable lessons about integrating physical computing with digital media. The challenge of synchronizing multiple systems in real-time pushed me to explore efficient communication protocols and optimize performance.

I found that physical interfaces provide an intuitive entry point for users to engage with digital art. There's something fundamentally satisfying about turning a physical knob and seeing immediate visual feedback that purely screen-based interactions can't match.

The most challenging aspect was ensuring stability across all systems while maintaining responsiveness. Finding the right balance between visual complexity and performance required careful testing and refinement.

Moving forward, I'm excited to explore more complex interactions between physical and digital elements, and to continue developing immersive experiences that bridge these worlds.
